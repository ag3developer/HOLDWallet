# Instalação do QR Code Scanner

Execute o comando abaixo para instalar a biblioteca de scanner de QR Code:

```bash
cd /Users/josecarlosmartins/Documents/HOLDWallet/frontend
npm install html5-qrcode
```

Ou se preferir yarn:

```bash
yarn add html5-qrcode
```

Esta biblioteca permite escanear QR Codes usando a câmera do dispositivo.
